Portfolio

Proyek Terbaru

Senang sekali bisa berbagi tentang proyek terbaru saya! Saat ini, saya tengah terlibat dalam proyek yang sangat menarik dan penuh tantangan. Proyek ini merupakan kombinasi antara kreativitas, inovasi, dan dedikasi yang tinggi.

G-MOOC 4D

G-MOOC 4D merupakan sebuah aplikasi pembelajaran untuk membantu penyandang disabilitas tunanetra dengan fitur speech recognition, face recognition, dan gamifikasi. Aplikasi ini dibuat secara team dan saya mengerjakan pada bagian model klasifikasi perintah suara (Speech recognition) menggunakan algoritma Artificial Neuaral Network (ANN) untuk frontendnya menggunakan React.Js dan backendnya menggunakan Laravel.

SwiftCare Kesehatan Mental

SwiftCare merupakan sebuah aplikasi chatbot berbasis website yang digunakan untuk bertanya informasi terkait kesehatan mental. Aplikasi ini dibuat secara team dan saya mengerjakan bagian modelling chatbotnya menggunakan algoritma Stochastic Gradient Descent (SGD) menggunakan python, flask untuk backend, dan forntend menggunakan tailwind.

SMS Guard Spam Deteksi

SMS Guard merupakan sebuah aplikasi berbasis website yang digunakan untuk mendeteksi spam sms apakah termasuk sms penipuan, sms normal, atau sms promo. Proyek ini dibuat menggunakan python untuk algoritmanya, untuk backend menggunakan flask, dan frontend menggunakan bootstrap. Aplikasi ini dibuat secara team dan saya mengerjakan bagian model klasifikasi Spam SMSnya

Search Movie

Search Movie adalah sebuah website yang digunakan untuk mencari sebuah film dengan mencari nama judulnya. website ini dibuat menggunakan framework boostrap, JavaScript, dan Jquery

Kusuma Craft

Kusuma Craft merupakan sebuah landing pages penjual jasa buket dan giftbox yang dibuat menggunakan framework bootstrap dan JavaScript

Firasat Coffe

Firasat Coofe Merupakan sebuah landing pages penjual coffe yang dibuat menggunakan framework bootsrap, dan jquery.
